Tower Defense for TF2C
"Tower Defense for TF2C" is a TF2C plugin that adds a (not at all) new gamemode, allowing you to place towers to defend yourself against the incoming waves of bots trying to cap your point.
View Repository
Angel's Time Machine
"Angel's Time Machine" is a TF2 plugin that adds a time machine command to freely switch the year that Team Fortress 2 takes place in. Changing the year can change what items you can use, along with their attributes.
View Repository
Old/Discontinued
Tri-Fortress
"Tri-Fortress" is a plugin for TF2 that adds a third team to the game. This has been discontinued due to lack of interest and mapping complications.
BWR Remastered
"BWR Remastered" is a plugin for TF2 that allows you to join the ROBOT team in MvM with the latest features and stability.
Smart Bots
"Smart Bots" is a plugin for TF2 that adds bots smarter than the ones that come prebundled with the game. Smart Bots will also support extra gamemodes (e.g. MvM).
AFK Bot Control
"AFK Bot Control" is a plugin for TF2 that allows bots to control your player while you're AFK. This is useful as you won't be disconnected for being AFK, and the team doesn't lose a player.
